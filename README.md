# 💼 Portfólio - José Luiz

Este é meu **site de portfólio pessoal**, desenvolvido para apresentar minhas informações, habilidades e links profissionais.

O projeto foi construído utilizando **HTML e CSS**, com foco em **organização de código, boas práticas e design responsivo**.

---

# 🚀 Tecnologias utilizadas

* 🌐 **HTML5**
* 🎨 **CSS3**
* 🔤 **Google Fonts**
* 📱 **Responsividade com Media Queries**

# 📱 Responsividade

O site foi desenvolvido para funcionar em **desktop e dispositivos móveis**, utilizando **Media Queries**.

```css
@media (max-width: 1200px) {
    .apresentacao {
        flex-direction: column-reverse;
    }
}
```
